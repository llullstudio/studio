Bordered mono uppercase button — EXTRAPOLATED from the chip/border language; the live site itself has no buttons. Use for prototypes needing actions.

```jsx
<Button>View Gallery</Button>
<Button variant="amber">Read on Substack</Button>
<Button variant="ghost">Map</Button>
<Button size="sm">Small</Button>
<Button disabled>Coming</Button>
```

Notes: square corners, transparent fill; amber variant hovers to a faint amber fill; ghost is borderless text that hovers amber.
